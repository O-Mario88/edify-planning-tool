## What to show (top demo moments)

- **Live role-switching** — one product reshapes for CCEO / PL / CD / RVP / Accountant / IA / HR / Partner.
- **School intake → auto-distribution** — uploaded schools appear instantly in the owner's portfolio; duplicates flagged (not blocked); unmatched owners mapped in a queue.
- **Partner delegation that keeps ownership** — assign a partner to deliver; the school never leaves the staff portfolio or targets.
- **Targets by time period** — fiscal-year ladder (Q1 25% → Q4 100%, mid-year), partner-supported schools included, pace-aware.
- **Operating-targets cockpit** — 7-period donut tiles + cumulative trend + target-by-period matrix.
- **Verified-only impact** — donor reporting counts only IA-verified, evidence-backed, de-duplicated work.
- **Finance cockpit** — fund approvals, disbursement queue, accountability, receipts — role-aware on one path.
- **Premium shell** — unified Edify branding, command palette (⌘K), three themes, photo-frame profiles.
